claude-code-booster は AI コーディングエージェント向けの開発加速基盤（ブースターキット）。スキル体系33定義、リファレンスドキュメント26ファイル、XP ロール定義9エージェント、テンプレート13ファイルで構成。v1.13.0（2026年3月時点）。

**Why:** AI エージェントの能力は与えるコンテキストの質に依存するため、ベストプラクティスを体系化してパッケージ配布することで組織の開発ベースラインを引き上げる狙い。

**How to apply:** 戦略的判断の際は「実案件での検証と効果測定」「段階的導入パスの設計」「フィードバックループの仕組み化」を優先事項として助言する。開発スキル（現在2つ）の拡充も投資対効果が高い領域として推奨。
